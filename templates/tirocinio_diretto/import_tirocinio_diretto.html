Importa Tirocini Diretti da Excel

Istruzioni

Il file Excel deve contenere le seguenti colonne nell'ordine specificato:

  • Studente (obbligatorio, formato: Nome Cognome) - Es: "Mario Rossi"
  • Scuola (obbligatorio) - Es: "Liceo Scientifico G. Galilei"
  • Tutor Esterno (obbligatorio) - Es: "Prof.ssa Anna Bianchi"
  • Data (obbligatorio, formato: GG/MM/AAAA) - Es: "25/01/2025"
  • Ore (obbligatorio, formato numerico) - Es: "4.5"
  • Descrizione Attività (obbligatorio) - Dettaglio delle attività svolte
